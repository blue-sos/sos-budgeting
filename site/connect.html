This endpoint is used as the public connect/reconnect URL for Intuit app metadata. The operational OAuth flow is initiated by an internal operator from local tooling.

Flow Summary

  1. Operator initiates OAuth against Intuit consent.
  2. Intuit redirects to callback with code and realmId.
  3. Credentials are exchanged and stored in local secure config.
