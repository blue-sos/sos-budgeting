This endpoint is provided for Intuit app metadata. SOS Budgeting is run as a local internal tool.

How Authorization Is Started

  1. An authorized operator opens the Intuit OAuth consent URL.
  2. After consent, Intuit redirects to the configured callback URL.
  3. The callback page displays code and realmId for local token exchange.

Operational Note

No credentials are entered on this page and no QuickBooks data is processed directly by this website.
