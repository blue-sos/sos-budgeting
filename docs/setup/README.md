# Setup Workspace

Owner: @platform
Last Reviewed: 2026-02-20
Status: active

This directory tracks human-in-the-loop setup status for a cloned project.

## Files

- `PREFLIGHT_STATUS.md`: checkbox state for required human steps.
- `EXTERNAL_TOOLING.md`: notes on MCP servers, credentials, and local tool endpoints.

Keep these files current so agents can reason about what is configured and what is still blocked.
