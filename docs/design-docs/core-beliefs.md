# Core Beliefs

Owner: @platform
Last Reviewed: 2026-02-20
Status: active

## Agent-First Beliefs

- Human attention is the scarce resource.
- Repository-local context is the only durable context for agents.
- Constraints and feedback loops scale better than manual intervention.
- Mechanical enforcement beats informal guidance.
- Incremental correction beats blocked throughput.

## Execution Beliefs

- Work depth-first: unlock capabilities, then compound.
- Prefer boring, legible abstractions over opaque complexity.
- Encode recurring review comments into docs, policy, or lints.
- Keep plans and decisions versioned next to code.
